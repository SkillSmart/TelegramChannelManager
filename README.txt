Hi Guys and Girls!

Please set up your Application like this:

1) Go and register at my.telegram.org
2) Open /app/config_copy.py
3) Insert the information from your API Registration page at my.telegram.org here

Then run the application:
1) Open your  Command Prompt in the root directory
2) Type: python setup.py install
3) Type: python run.py

Then follow the instructions in the Command line interface:
1) Choose "Bann users from a given channel"
2) Choose to create a new Index: Y
2) Keep the standards
